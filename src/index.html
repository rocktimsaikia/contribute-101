Contribue-101

Contribute-101 is an initiative designed to guide new developers in making their first open source contribution. Our goal is to help beginners become comfortable with open source projects and get familiar with the standard GitHub workflow.

Check out our GitHub Repository for detailed instructions and a step-by-step guide on contributing. While you're there, show some love by starring the repo! 💖 Happy contributing!

This project is maintained by @utsah. Previously @rocktim.
